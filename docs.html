准备工作

欢迎来到科成MC服务器！在开始游戏前，您需要完成一些准备工作。

账号准备

科成MC服务器需要正版账号或外置登录账号：

  1. 如果您有Minecraft正版账号，可以直接使用
  2. 如果没有正版账号，您需要注册我们的外置登录账号：
    • 访问皮肤站注册账号
    • 或访问注册站注册账号
  3. 注册完成后，请妥善保管您的账号信息

启动器准备

您需要使用支持外置登录的启动器：

  • PCL2启动器：
    1. 下载并安装PCL2启动器
    2. 在设置中找到「验证服务器」选项
    3. 选择「第三方登录（authlib-injector）」
    4. 填写认证服务器：https://skin.cduestc.fun/api/yggdrasil
    5. 点击「保存」
  • HMCL启动器：
    1. 下载并安装HMCL启动器
    2. 点击「账户」-「添加账户」
    3. 选择「外置登录（authlib-injector）」
    4. 填写认证服务器：https://skin.cduestc.fun/api/yggdrasil
    5. 输入您在皮肤站注册的用户名和密码

注意：请确保您的Minecraft版本兼容服务器（支持1.8-1.21版本）。推荐使用1.21版本获得最佳体验。

初入游戏

完成账号和启动器设置后，您可以开始进入服务器并进行简单配置。

服务器简介

科成MC是电子科技大学成都学院（CDUESTC）的Minecraft公益服务器，由科成MC同好会成员联合创办。我们提供多种类型的服务器，包括原版生存、整合包等，满足不同玩家的需求。

如何加入服务器

  1. 加入QQ群：957464722
  2. 在群公告中获取服务器地址和相关信息
  3. 启动游戏，选择对应版本（推荐1.21）
  4. 点击「多人游戏」-「添加服务器」
  5. 输入服务器地址并保存
  6. 选择服务器并点击「加入服务器」

初次登录设置

首次进入服务器时，您需要完成以下设置：

  1. 按照服务器提示设置密码（如果使用外置登录则无需此步骤）
  2. 阅读服务器规则并同意
  3. 使用/register <密码> <确认密码>注册（如需要）
  4. 使用/login <密码>登录（如需要）

基本设置

进入服务器后，建议进行以下基本设置：

  • 使用/sethome设置您的家，方便日后传送
  • 使用/nick设置您的昵称（如果服务器支持）
  • 使用/tps查看服务器性能状态
  • 使用/bal或/money查看您的余额

注意：首次进入服务器建议先熟悉周围环境，阅读服务器公告，了解基本规则和指令。

服务器规则

为了维护良好的游戏环境，请所有玩家遵守以下规则：

基本规则

  • 尊重其他玩家，禁止辱骂、歧视等不文明行为
  • 禁止使用任何作弊客户端、模组或外挂
  • 禁止恶意破坏其他玩家的建筑和财产
  • 禁止利用游戏漏洞或服务器漏洞获取不正当利益
  • 禁止发布违反法律法规的内容

建筑规则

  • 请在合适的地点建造，不要过于靠近其他玩家
  • 禁止建造过于巨大的红石机器导致服务器卡顿
  • 禁止建造不良内容的建筑

警告：违反服务器规则可能会导致临时或永久封禁。

简单指令

以下是服务器中常用的指令，帮助您更好地游玩：

基础指令

/help - 查看帮助信息
/spawn - 返回出生点
/tpa <玩家名> - 请求传送到指定玩家身边
/tpaccept - 接受传送请求
/tpdeny - 拒绝传送请求
/sethome - 设置家
/home - 传送回家
/back - 返回上一个位置（如死亡地点）
/msg <玩家名> <消息> - 私聊玩家
/r <消息> - 回复最后私聊你的玩家

经济指令

/balance 或 /bal - 查看余额
/pay <玩家名> <金额> - 向指定玩家转账
/baltop - 查看服务器财富排行榜

领地指令

/res create <领地名> - 创建领地
/res info - 查看当前所在领地信息
/res set <权限> <true/false> - 设置领地权限
/res tp <领地名> - 传送到指定领地
/res list - 查看自己的领地列表
/res padd <领地名> <玩家名> - 添加玩家到领地

特色指令

/tps - 查看服务器性能
/hat - 将手中物品戴在头上
/afk - 标记自己为暂时离开
/mail read - 阅读邮件
/mail send <玩家名> <内容> - 发送邮件
/nick <昵称> - 设置昵称（如支持）

提示：使用 /help <插件名> 可以查看特定插件的帮助信息。服务器会不定期更新指令，请关注服务器公告。

进阶配置

熟悉基本操作后，您可以进行一些进阶配置，提升游戏体验。

玩家选项配置

服务器提供了丰富的玩家自定义选项，您可以根据自己的喜好进行设置：

  • 聊天格式：可以自定义聊天消息的显示格式
  • 物品显示：可以调整物品信息的显示方式
  • 游戏提示：可以开启或关闭各类游戏提示
  • 界面设置：可以自定义界面元素的显示

锻造系统进阶

服务器的锻造系统非常丰富，进阶使用方法：

  1. 收集稀有材料，用于高级装备的锻造
  2. 了解不同附魔的特性和适用场景
  3. 合理搭配装备属性，提升整体战斗力
  4. 使用特殊锻造台，制作专属装备

红石与自动化

服务器支持大部分红石功能，您可以建造自动化设施：

  • 自动农场：提高资源收集效率
  • 自动分类系统：管理物品存储
  • 自动交易系统：便捷的物品交换
  • 注意：请合理控制红石设备规模，避免造成服务器卡顿

社区参与

深度参与服务器社区活动，获得更好的游戏体验：

  • 参与服务器举办的各类活动和比赛
  • 加入玩家组建的城镇或公会
  • 在社区论坛或QQ群中分享游戏心得
  • 为服务器发展提供建议和反馈

插件进阶使用

熟练使用服务器的特色插件功能：

  • 使用经济插件建立自己的商店
  • 利用领地插件创建复杂的权限系统
  • 探索任务插件提供的高级任务链
  • 使用投票奖励插件获取额外福利

注意：进行高级操作时，请确保了解相关规则，避免因误操作导致损失或违规。如有疑问，请随时咨询管理员。

经济系统

服务器采用虚拟货币系统，您可以通过多种方式获取和使用货币。

获取货币

  • 完成任务和成就
  • 在商店出售物品
  • 参与服务器活动
  • 与其他玩家交易

使用货币

  • 购买物品和服务
  • 创建和扩展领地
  • 参与拍卖
  • 支付传送费用

商店系统

服务器提供多种商店，您可以在这些商店中购买或出售物品：

  • 系统商店：提供基础物品的买卖
  • 玩家商店：由其他玩家创建的商店
  • 特殊商店：提供稀有物品和特殊服务

领地保护

服务器使用领地插件保护您的建筑和财产不被破坏。

创建领地

  1. 使用木锄选择领地的两个对角点（左键选择第一个点，右键选择第二个点）
  2. 输入 /res create <领地名> 创建领地
  3. 创建领地需要支付费用，费用与领地大小成正比

领地权限

您可以为领地设置各种权限，控制其他玩家在您领地内的行为：

  • build - 建造权限
  • destroy - 破坏权限
  • use - 使用物品权限
  • container - 使用容器权限
  • pvp - PVP权限

添加信任

您可以将其他玩家添加为领地的信任成员，允许他们在您的领地内进行特定操作：

/res padd <领地名> <玩家名> - 添加玩家到领地
/res pdel <领地名> <玩家名> - 从领地移除玩家

锻造系统

服务器拥有特色的锻造系统，允许玩家制作和升级特殊装备。

基础锻造

在主城的铁匠铺可以进行基础锻造，制作特殊装备：

  1. 收集所需材料
  2. 与铁匠NPC对话
  3. 选择要锻造的装备类型
  4. 支付相应的费用

装备升级

已有的装备可以通过升级系统提升属性：

  • 使用特殊材料和货币升级装备
  • 升级有一定的成功率，失败可能导致装备降级或破损
  • 高级装备需要特殊的升级材料

附魔系统

服务器拥有扩展的附魔系统，提供更多的附魔选项：

  • 自定义附魔可以提供特殊效果
  • 某些附魔需要特殊的附魔书
  • 附魔等级可以超过原版限制

插件功能

服务器使用多种插件增强游戏体验，以下是一些主要插件的功能介绍：

经济插件

提供完整的经济系统，包括货币、商店、拍卖等功能。

领地插件

允许玩家创建和管理自己的领地，保护建筑和财产。

传送插件

提供各种传送功能，包括传送到玩家、设置家等。

任务插件

提供各种任务和成就，完成后可获得奖励。

聊天插件

增强聊天功能，支持私聊、频道、彩色文字等。

常见问题

如何注册账号？

首次进入服务器时，系统会提示您注册。按照提示输入密码即可完成注册。

忘记密码怎么办？

请联系服务器管理员重置密码。

服务器支持哪些版本？

服务器支持Java版1.8-1.21的客户端进入。

如何赚取游戏币？

可以通过完成任务、出售物品、参与活动等方式赚取游戏币。

遇到恶意玩家怎么办？

请使用 /report <玩家名> <原因> 举报恶意玩家，管理员会尽快处理。

联系我们

QQ群

加入我们的QQ群：957464722

服务器内联系

反馈建议

注意：管理员不会要求您提供账号密码，请注意账号安全。
